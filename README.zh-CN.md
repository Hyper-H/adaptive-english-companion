# Adaptive English Companion

[简体中文](README.zh-CN.md) | [English](README.md)

一个面向 Codex 的英语学习 skill，目标不是把你训练成做题机器，而是给你一个会越来越熟悉你的“陪伴式英语老师”。

`Adaptive English Companion` 想做的，不是传统那种一板一眼的纠错器，而是一个更像熟悉你的老师的英语搭子。它支持中英混输，会优先理解你真正想表达的意思，再给出自然英文表达；它会根据你的理解情况决定哪里需要中文、哪里可以直接英文推进；如果环境支持保存学习者信息，它还会随着使用逐渐变得更懂你。

## 它的特点

- 更像陪伴式老师，不像考试批改器
- 允许你先说出来，不要求一开始就全英文完美输出
- 优先理解你的真实意思，而不是只盯着语法表面错误
- 更偏向给自然表达，而不是动不动就上长篇语法课
- 当你明显能跟上时，会减少不必要的中文解释
- 会随着使用逐渐更熟悉你的习惯、弱点和偏好

## 你实际会怎么用它

安装完之后，理想体验其实很简单：

1. 直接开始聊天
2. 把它当英语老师来用
3. 个性化能力交给 agent 自动处理

你不需要先手动维护什么配置文件，skill 本身就应该先能用起来。

如果当前环境支持保存学习者信息，agent 会自动在后台维护一个很小的 learner profile；如果当前环境不支持保存，它也仍然可以正常作为一个自适应英语教练使用。

## 它能做什么

- 自然接收中英混合输入
- 先推断你真正的意思，再决定怎么纠正
- 主动给出更自然的英文表达
- 只有在歧义真的明显时才额外确认
- 根据你的理解情况动态决定中文解释多少
- 既适合日常表达，也适合学习交流和科研讨论
- 在环境允许时，建立轻量的长期个性化能力

## 快速开始

安装好 skill 之后，直接这样开聊就行：

- `Use $adaptive-english-companion to be my English teacher.`
- `English teacher, help me practice through mixed Chinese-English conversation.`
- `英语老师，和我一起练英语。`
- `ET: help me express this idea naturally in English.`

到这里就够了。用户不需要额外手动配置。

## 推荐触发方式

最稳的写法：

- `$adaptive-english-companion`

更轻便的写法：

- `English teacher`
- `英语老师`
- `ET:`

如果是新开对话，完整写 skill 名最稳。日常使用时，短触发词通常会更方便。

## 自动个性化

这个 skill 的核心特色之一，就是“轻量个性化”。

它不要求用户自己长期维护学习档案。更理想的流程应该是：

- 用户安装 skill
- 用户直接开始聊天
- agent 逐渐观察稳定的学习特征和偏好
- agent 随着使用变得越来越熟悉这个用户

如果环境支持保存学习者信息，skill 会自动维护一个很短的 learner profile。这个 profile 不是聊天记录，也不是每句错误清单，而是一份很短的工作摘要，比如：

- 当前大致水平
- 重复出现的错误模式
- 偏好的纠错方式
- 当前重点想练的内容

它应该保持很短。它的目标是让老师更熟悉你，而不是生成一份越来越长的学习报告。

## 常见问题

### 安装后能直接用吗？

可以。安装后直接开始聊天就行。

### `learner-profile.md` 需要我自己创建吗？

不需要。这个 skill 的设计目标就是：用户不需要先手动创建个人配置，装上就能开始用。

### “稳定模式”是谁判断的？

是 agent 自己判断。

这里的“稳定模式”，指的是重复出现或者高置信度的学习者特征，而不是一次性的偶然小错误。

通常算稳定模式的情况：

- 长句经常出现意思漂移
- 经常把中文表达逻辑直接搬进英文
- 经常偏好轻纠错、少打断
- 对抽象内容更常需要中文辅助

通常不该记成稳定模式的情况：

- 一次拼写错误
- 一句不够自然
- 一次临时时态用错

### 会不会很耗 token、让响应变慢？

正常使用时，一般不会特别重。

真正容易增加 token 和响应时间的，通常是这些情况：

- 长段内容同时完整输出中英两份
- 每一轮都先确认意思再继续
- learner profile 写得太长
- 每条消息都更新 profile
- 每次纠错都展开成完整语法课

这个 skill 默认就是按轻量思路设计的：

- 中文解释是自适应的，不是每句必带
- learner profile 保持简短
- 只有出现稳定模式时才更新
- 更偏向短而有价值的自然改写，而不是长篇分析

实际使用里，真正最容易费 token 的，通常不是 skill 本身，而是长篇双语重复输出。

## 适合谁

- 不喜欢“先背后用”英语学习方式的人
- 会自然中英混着想、混着说的中文使用者
- 既想练日常英语，也想练学术或科研英语的学生
- 希望有一个温和、熟悉、陪伴式英语老师的人
