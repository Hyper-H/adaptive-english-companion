# English Learner Profile 模板

把它当成放在 skill 根目录 `learner-profile.md` 的轻量学习者画像。

保持简短、可修改、以长期模式为主。

低 token 建议：

- 整个文件尽量控制在 8 到 15 条 bullet 左右。
- 大多数区块保持在 0 到 2 条 bullet。
- 优先写短词组或短句，不写长段落。
- 真正生成 profile 时，把空的占位 bullet 删除。
- 如果某个标题暂时没有长期有用的信息，可以整个标题先省略。
